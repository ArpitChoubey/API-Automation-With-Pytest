# 🎯 Project Overview

Modern applications are powered by APIs.

If an API fails, it directly impacts customer experience, business workflows, mobile applications, and integrated systems.

This repository demonstrates how API quality can be validated using **Python**, **Pytest**, and **REST API Automation** by implementing reusable test scripts, modular architecture, and scalable automation practices.

Instead of validating only HTTP status codes, this project focuses on verifying functional correctness, response validation, reusable automation, and maintainable test design.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/ArpitChoubey/API-Automation-With-Pytest.git
```

---

## Navigate

```bash
cd API-Automation-With-Pytest
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

or

```bash
pip install pytest
pip install requests
pip install pandas
```

---

## Execute Tests

Run all tests:

```bash
pytest
```

Run a specific module:

```bash
pytest API_Test05.py
```

---

## Generate Allure Report

```bash
pytest --alluredir=allure_result
```

Generate report:

```bash
allure serve allure_result
```
